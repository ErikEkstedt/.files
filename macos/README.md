# MacOs

### Neovim:

Download tar, save in home directory and link ~/.local/bin/nvim

´´´bash
cd ~
https://github.com/neovim/neovim/releases/download/nightly/nvim-macos.tar.gz
tar xzf nvim-macos.tar.gz
./nvim-osx64/bin/nvim

ln -s nvim-osx64/bin/nvim ~/.local/bin/nvim
´´´
